## Given

A *real-time $\log n$ depth bounded iterative tree array* $M$ and an input string $x$ of length $n$.

## Problem

Does $M$ accept $x$?

## Definitions

See Ibarra, Jiang, and Chang <a href="#ref-1" class="reference-citation">[1]</a>.

## Status

$\P$-hard (Ibarra, Jiang, and Chang <a href="#ref-1" class="reference-citation">[1]</a>).

## Remarks

Iterative tree arrays can simulate cellular automata, so the reductions is from [CTAA]({{< relref "./a-7-12.md" >}}).
Note that under the appropriate resource bounds cellular tree automata can simulate iterative tree arrays <a href="#ref-1" class="reference-citation">[1]</a>.
