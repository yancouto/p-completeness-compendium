## Given

A real-time $\log n$ depth bounded iterative tree array $M$ and an input string $x$ of length $n$. See Ibarra, Jiang, and Chang [165] for definitions.

## Problem

Does $M$ accept $x$?

## Status

P-complete problem from Greenlaw et al. (1995), Problem A.7.13.

## Remarks

Note that under the appropriate resource bounds cellular tree automata can simulate iterative tree arrays [165]. Also see Problem A.7.12.
