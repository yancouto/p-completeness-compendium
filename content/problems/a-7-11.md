## Given

Let $S$ be a set, and $S_o$ and $S_f$ elements of $S$. A set system consists of a finite set of productions $P_S$ of the form $X \to Y$, where $X$ and $Y$ are subsets of $S$. A set $B \subseteq S$ derives in one step a set $C \subseteq S$, written $B \Rightarrow C$, if and only if $C = \bigcup_{X \to Y \in P_S, X \subseteq B} Y$. $\stackrel{*}{\Rightarrow}$ denotes the reflexive, transitive closure of $\Rightarrow$. A propagating set system is a set system such that for every element $s \in S$, $\\{s\\} \to \\{s\\} \in P_S$.

## Problem

Does $S_o \stackrel{*}{\Rightarrow} S'$ such that $S' \cap S_f \neq \varnothing$?

## Status

$\P$-complete (Monti and Roncato [1]).

## Remarks

The general version of the problem is PSPACE-complete.
