## Given

A Boolean formula $F$ with clauses $c_1, \ldots, c_m$ and variables $x_1, \ldots, x_n$, plus an integer $l$.

## Problem

Does variable $x_l$ get assigned a value true by the maximum satisfiability approximation algorithm? The algorithm is sketched below. Its input is the same as for MSA.

```
begin
  for i ← 1 to n do
    for j ← 1 to m do
      f_T(i, j) ← the fraction of truth assignments for the remaining
                    variables of clause j that satisfy clause j with x_i
                    set to true
      f_F(i, j) ← 1 − f_T(i, j)
    if ∑_{j=1}^{m} f_T(i, j) ≥ ∑_{j=1}^{m} f_F(i, j) then
      x_i ← true
    else
      x_i ← false
    update the clauses
end
```

## Status

$\P$-complete problem from Greenlaw et al. (1995), Problem A.5.10.

## Remarks

The algorithm specified above $1/2$ approximates MSA. The result shows that this sequential algorithm for approximating Maximum Satisfiability is unlikely to be made feasibly highly parallel.
