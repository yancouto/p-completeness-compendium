## Given

An $n$-vertex polygon $P$ without holes.

## Problem

Find the triangulation computed by the *plane sweep triangulation algorithm*.

## Status

Open (Atallah, Callahan, and Goodrich <a href="#ref-2" class="reference-citation">[2]</a>).

## Definitions

See [PST]({{< relref "./a-9-1.md" >}}) for a description of the algorithm.

## Remarks

The problem of finding some arbitrary triangulation is in $\NC$ (Goodrich <a href="#ref-1" class="reference-citation">[1]</a>). If the polygon is allowed to have holes, the problem is $\P$-complete <a href="#ref-2" class="reference-citation">[2]</a>. See [PST]({{< relref "./a-9-1.md" >}}).
