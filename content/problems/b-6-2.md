## Given

An $n$-vertex polygon $P$ without holes.

## Problem

Find the triangulation computed by the plane sweep triangulation algorithm. See Problem A.9.1 for a description of the algorithm.

## Status

Open problem from Greenlaw et al. (1995), Problem B.6.2.

## Remarks

The problem of finding some arbitrary triangulation is in $\NC$ (Goodrich [1]). If the polygon is allowed to have holes, the problem is $\P$-complete [2]. See Problem A.9.1.
