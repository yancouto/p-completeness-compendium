## Given

A three layered network $G = (V, E)$ with each edge labeled with a capacity $c_i \geq 0$, and two distinguished vertices, source $s$ and sink $t$. A three layered network is one in which all source to sink paths have length three.

## Problem

Find a blocking flow. A blocking flow is a flow in which every path from $s$ to $t$ has a saturated edge — an edge whose flow is equal to its capacity.

## Status

Open problem from Greenlaw et al. (1995), Problem B.9.1.

## Remarks

The problem is in $\text{RNC}$ [54]. The problem of finding the lexicographically first blocking flow in a three layered network is $\text{P}$-complete (Anderson and Mayr [15]). See Problem A.4.6. The problem of finding a blocking flow in an acyclic network is also open [120]. In this case Goldberg and Tarjan give an EREW-PRAM algorithm that uses $O(n \log n)$ time and $m$ processors for an $n$-vertex, $m$-edge network [120].
