## Given

A set $\Sigma$ of multivalued dependencies and a multivalued dependency $\sigma$. We follow the discussion presented by Denenberg in [1] to define these concepts. Let $U$ denote the set of all attributes in the database and let $X$, $Y$, and $Z$ denote sets of attributes. An embedded multivalued dependency (EMVD) is an expression of the form $X \to\to Y \mid Z$, where $X$, $Y$, and $Z$ are disjoint sets of attributes in which $Y$ and $Z$ nonempty. A multivalued dependency (MVD) is an EMVD in which $Z$ is equal to $U - XY$. A set $\Sigma$ of MVDs implies a MVD $\sigma$ if $\sigma$ holds in every database in which every member of $\Sigma$ holds.

## Problem

Does $\Sigma$ imply $\sigma$?

## Status

$\P$-complete problem from Greenlaw et al. (1995), Problem A.6.5.
