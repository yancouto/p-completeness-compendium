## Given

A context-free grammar $G = (N, T, P, S)$ and a string $x \in T^\ast$.

## Problem

Is $x \in L(G)$?

## Definitions

Here, $N$ is the set of *nonterminal symbols*, $T$ is the set of *terminal symbols*, $P$ is the set of *production rules*, and $S$ is the *start symbol*.

## Status

$\P$-complete (Jones and Laaser <a href="#ref-1" class="reference-citation">[1]</a>).

## Remarks

The reduction to show hardness is from [GEN]({{< relref "./a-6-7.md" >}}).
Goldschlager remarks it is the presence of $\epsilon$-productions in the input grammar that make the membership question difficult <a href="#ref-2" class="reference-citation">[2]</a>. Lewis, Stearns, and Hartmanis' $(\log n)^2$ space algorithm <a href="#ref-3" class="reference-citation">[3]</a> and Ruzzo's $\mathsf{AC}^1$ (hence $\NC^2$) algorithm <a href="#ref-4" class="reference-citation">[4]</a> for general context free language recognition can both be modified to work with an $\epsilon$-free grammar given as part of the input. For a fixed grammar, unbounded fanin size $O(n^6)$ suffices <a href="#ref-4" class="reference-citation">[4]</a>.

Rytter shows context-free languages can be recognized on a cube connected computer in $O((\log n)^2)$ time using $n^6$ processors <a href="#ref-5" class="reference-citation">[5]</a>. He also provides an $O(\log n)$ time $n^7$ processor COMMON CRCW- PRAM algorithm for the recognition of unambiguous context-free languages <a href="#ref-6" class="reference-citation">[6]</a>. Klein and Reif give an $O(\log n)$ time $n^3$ processor CREW-PRAM algorithm for the recognition of deterministic context-free languages <a href="#ref-7" class="reference-citation">[7]</a>. Dymond and Ruzzo give a somewhat simpler algorithm for this problem on the more restricted CROW- PRAM <a href="#ref-8" class="reference-citation">[8]</a>. Kaji et al. prove *Parallel Multiple Context-free Grammar Membership* and *Multiple Context-free Grammar Membership*, which are problems involving generalizations of context-free grammars that are designed to model natural languages, are $\P$-complete <a href="#ref-9" class="reference-citation">[9]</a>.
