## Given

An undirected graph $G = (V, E)$ with an ordering on the vertices and a designated vertex $v$.

## Problem

Is vertex $v$ in the lexicographically first maximal independent set of $G$?

## Status

$\P$-complete under $\NC^1$ reductions (Greenlaw, Roover and Ruzzo [10, Theorem 7.1.2], Cook <a href="#ref-1" class="reference-citation">[1]</a>).

## Remarks

This is an instance of the [LFMS(π)]({{< relref "./a-2-16.md" >}}). LFMIS is $\P$-complete for bipartite or planar graphs restricted to degree at most three <a href="#ref-2" class="reference-citation">[2]</a>. Karp observed that the completeness of LFMIS implies that determining the $i^\text{th}$ vertex chosen by any deterministic sequential algorithm for either LFMIS or LFMC is also complete <a href="#ref-3" class="reference-citation">[3]</a>. Computing or approximating the size of the lexicographically first maximal independent set is also $\P$-complete [10, Section 10].

Luby gave the first $\NC$ algorithm for finding a maximal independent set <a href="#ref-4" class="reference-citation">[4]</a>, subsequently improved by Luby <a href="#ref-5" class="reference-citation">[5]</a>, by Alon, Babai, and Itai <a href="#ref-6" class="reference-citation">[6]</a>, and by Goldberg and Spencer <a href="#ref-7" class="reference-citation">[7]</a>. These algorithms do not compute the *lexicographically first* maximal independent set. Using a general result involving inference systems, when $G$ is a forest LFMIS can be solved on a CREW-PRAM in $O(\log |V|)$ time using a polynomial number of processors <a href="#ref-9" class="reference-citation">[9]</a>.
