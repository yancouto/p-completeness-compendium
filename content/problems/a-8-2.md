## Given

A stationary Markov decision process $M = (S, c, p)$ and a real number $\beta \in (0, 1)$. See Problem A.8.1 for definitions. This problem is infinite horizon, that is, there is no time bound.

## Problem

Is the minimum expected cost of $\sum_{t=0}^{\infty} c(s_t, \delta(s_t, t)) \beta^t$ over all policies $\delta$ equal to $0$?

## Status

$\P$-complete problem from Greenlaw et al. (1995), Problem A.8.2.

## Remarks

The Infinite Horizon, Discounted, Deterministic Problem is in $\text{NC}$ [1]. The deterministic problem requires that $p$ only has values $0$ or $1$.
