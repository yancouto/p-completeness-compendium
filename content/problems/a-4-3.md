## Given

An integer $n \times d$ matrix $A$, an integer $n \times 1$ vector $b$, and an integer $1 \times d$ vector $c$.

## Problem

Find a rational $d \times 1$ vector $x$ such that $Ax \leq b$ and $cx$ is maximized.

## Status

$\P$-hard (Dobkin, Lipton, and Reiss <a href="#ref-1" class="reference-citation">[1]</a>, Dobkin and Reiss <a href="#ref-2" class="reference-citation">[2]</a>, Khachian <a href="#ref-9" class="reference-citation">[9]</a>, Valiant <a href="#ref-10" class="reference-citation">[10]</a>).

## Remarks

The original reduction in <a href="#ref-1" class="reference-citation">[1]</a> is from [HORN]({{< relref "./a-6-2.md" >}}) to LP. In <a href="#ref-2" class="reference-citation">[2]</a>, LP and [LI]({{< relref "./a-4-1.md" >}}) are shown to be logarithmic space equivalent by reducing LP to [LI]({{< relref "./a-4-1.md" >}}) using *rational binary search* (Papadimitriou <a href="#ref-3" class="reference-citation">[3]</a>, Reiss <a href="#ref-4" class="reference-citation">[4]</a>) to find the value of the maximum and an $x$ that yields it. However, it is not clear how to perform this reduction in $\NC^1$. Since LP and [LI]({{< relref "./a-4-1.md" >}}) are complete via $\NC^1$ reductions though, there must be an $\NC^1$ reduction between the two problems. Although we know that LP and [LI]({{< relref "./a-4-1.md" >}}) are $\NC^1$ equivalent, the $\NC^1$ reduction between them is not an obvious one. Serna shows it is $\P$-complete to approximate a *solution approximation*, which requires finding an $x'$ close to the optimal solution, and to approximate a *value approximation*, which requires finding an $x'$ such that $cx'$ is close to $cx$ <a href="#ref-5" class="reference-citation">[5]</a>. Megiddo gives a reduction of LP to Serna's approximate problem <a href="#ref-6" class="reference-citation">[6]</a>. It is also $\P$-hard to approximate $cx$ to within any constant fraction, even given a feasible solution $x'$ <a href="#ref-11" class="reference-citation">[11]</a>; reduction is from [DMDP]({{< relref "./a-8-2.md" >}}). See Lin-Kriz and Pan <a href="#ref-7" class="reference-citation">[7]</a> for some results on two variable linear programming. Dahlhaus observes that for $d$ equal to 3 and values over the reals the problem is $\P$-complete, whereas for $d$ equal to 2, the problem is in $\NC$ <a href="#ref-8" class="reference-citation">[8]</a>.
