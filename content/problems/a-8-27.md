## Given

A set of total functions $a_1, \ldots, a_r$ and a designated function $f$, all from the set $\\{1, 2, \ldots, m\\}$ to itself, with the property that $\langle a_1, \ldots, a_r \rangle \in B_2$. $\langle a_1, \ldots, a_r \rangle$ denotes the set of all functions obtained by composing the functions $a_i$ with one another. $B_2$ is a restricted class of aperiodic monoids. The reader is referred to [29] for definitions.

## Problem

Does $f$ belong to $\langle a_1, \ldots, a_r \rangle$?

## Status

$\P$-complete problem from Greenlaw et al. (1995), Problem A.8.27.

## Remarks

The lattice of aperiodic monoid varieties is partitioned in [29] into the following five sublattices: $B_1$, $B_2$, $B_3$, $B_4$, and $B_5$. The membership problem for varieties in $B_2$ as described above is $\P$-complete. For varieties in $B_1$ the problem is in $\text{AC}^0$; for $B_3$ it is $\text{NP}$-complete; for $B_4$ it is $\text{NP}$-hard; for $B_5$ it is $\text{PSPACE}$-complete [29].
