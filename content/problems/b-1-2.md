## Given

A tree $T = (V, E)$.

## Problem

Find an optimal edge ranking of $T$. An edge ranking of a tree is a labeling of the edges using positive integers such that the path between two edges with the same label contains an intermediate edge with a higher label. An edge ranking is optimal if the highest label used is as small as possible.

## Status

Open problem from Greenlaw et al. (1995), Problem B.1.2.

## Remarks

This problem was proved to be in $\P$ in [1]. They give a $\NC$ approximation algorithm for the problem that finds an edge ranking within a factor of two of optimal. An $\NC$ algorithm for constant degree trees is also given in [1]. A similar problem called the Node Ranking Problem for trees in which the vertices are labeled, instead of the edges, was proved to be in $\NC$ (de la Torre and Greenlaw [2], de la Torre, Greenlaw, and Przytycka [3]). See Problem A.8.25 for a related $\P$-complete problem. Karloff and Shmoys give $\NC$ algorithms for several versions of edge ranking multigraphs [4], a problem having a similar flavor. See Problems A.2.6 and B.9.3 for a description of their results.
