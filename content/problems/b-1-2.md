## Given

A tree $T = (V, E)$.

## Problem

Find an optimal edge ranking of $T$. An edge ranking of a tree is a labeling of the edges using positive integers such that the path between two edges with the same label contains an intermediate edge with a higher label. An edge ranking is optimal if the highest label used is as small as possible.

## Status

Open problem from Greenlaw et al. (1995), Problem B.1.2.

## Remarks

This problem was proved to be in $\text{P}$ in [84]. They give a $\text{NC}$ approximation algorithm for the problem that finds an edge ranking within a factor of two of optimal. An $\text{NC}$ algorithm for constant degree trees is also given in [84]. A similar problem called the Node Ranking Problem for trees in which the vertices are labeled, instead of the edges, was proved to be in $\text{NC}$ (de la Torre and Greenlaw [81], de la Torre, Greenlaw, and Przytycka [82]). See Problem A.8.25 for a related $\text{P}$-complete problem. Karloff and Shmoys give $\text{NC}$ algorithms for several versions of edge ranking multigraphs [191], a problem having a similar flavor. See Problems A.2.6 and B.9.3 for a description of their results.
