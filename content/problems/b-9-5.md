## Given

An undirected graph $G = (V, E)$ and a set of vertices $U \subseteq V$.

## Problem

Find a maximum cardinality set of nontrivial vertex disjoint paths that have their endpoints in $U$.

## Status

Open (Anderson <a href="#ref-1" class="reference-citation">[1]</a>).

## Remarks

MDP is first reduced to a bidirectional flow problem that is in turn reduced to a matching problem <a href="#ref-1" class="reference-citation">[1]</a>. See Problem A.3.2.
