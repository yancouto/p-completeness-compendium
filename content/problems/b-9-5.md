## Given

An undirected graph $G = (V, E)$ and a set of vertices $U \subseteq V$.

## Problem

Find a maximum cardinality set of nontrivial vertex disjoint paths that have their endpoints in $U$.

## Status

Open problem from Greenlaw et al. (1995), Problem B.9.5.

## Remarks

MDP is first reduced to a bidirectional flow problem that is in turn reduced to a matching problem [1]. See Problem A.3.2.
