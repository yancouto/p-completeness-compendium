## Given

A set of $n$ men and a set of $n$ women. For each person a ranking of the opposite sex according to their preference for a marriage partner. Note that a preference list does not need to include a ranking for every member of the opposite sex. This problem is sometimes called the Stable Roommates Problem.

## Problem

Does the given instance of the problem have a set of marriages that is stable? The set is stable (or a set of stable marriages) if there is no unmatched pair $\\{m, w\\}$ such that both $m$ and $w$ prefer each other to their current partners.

## Status

Open (Mayr and Subramanian <a href="#ref-3" class="reference-citation">[3]</a>, P'olya, Tarjan, and Woods <a href="#ref-2" class="reference-citation">[2]</a>).

## Remarks

See, for example, Gibbons <a href="#ref-1" class="reference-citation">[1]</a> or <a href="#ref-2" class="reference-citation">[2]</a> for background on SM. If the preference lists are complete, the problem always has a solution <a href="#ref-2" class="reference-citation">[2]</a>. SM is $\NC$ equivalent to Comparator Circuit Value, Problem B.8.1 <a href="#ref-3" class="reference-citation">[3]</a>. Several variations of SM are also known to be equivalent to [CCVP]({{< relref "./b-8-1.md" >}}). For example, the Male-optimal Stable Marriage Problem in which there is a designated couple $\\{m, w\\}$ and the question asked is whether man $m$ is married to woman $w$ in the male-optimal stable marriage? <a href="#ref-3" class="reference-citation">[3]</a>. The male-optimal stable marriage is the one formed by the algorithm given in <a href="#ref-2" class="reference-citation">[2]</a>. It finds a matching in which no man could do any better in a stable marriage. Several other versions of SM are discussed in Problems B.8.4, B.8.5, and B.8.6.
