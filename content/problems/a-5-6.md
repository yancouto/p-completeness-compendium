## Given

An undirected graph $G = (V, E)$ with $2n$ vertices.

## Problem

Find a locally optimal partition of $V$ into two equal size sets.

The partition $V_1$ and $V_2$ is *locally optimal* if the number of edges going between $V_1$ and $V_2$ a minimum among all neighbors of the partition. A *neighbor* is a partition that can be obtained from $V_1$ and $V_2$ by swapping one element of $V_1$ with one element of $V_2$.

## Status

$\P$-complete (Papadimitriou, Schäffer, and Yannakakis <a href="#ref-1" class="reference-citation">[1]</a>, Savage and Wloka <a href="#ref-5" class="reference-citation">[5]</a><a href="#ref-4" class="reference-citation">[4]</a>, Schäffer and Yannakakis <a href="#ref-2" class="reference-citation">[2]</a>).

## Remarks

The weighted version of the problem in which the weights are encoded in binary is $\mathsf{PLS}$-complete <a href="#ref-1" class="reference-citation">[1]</a><a href="#ref-2" class="reference-citation">[2]</a>. If the weights are encoded in unary, the problem is $\mathsf{FP}$-complete (Johnson, Papadimitriou, and Yannakakis <a href="#ref-3" class="reference-citation">[3]</a>). Another problem the *Fiduccia-Mattheyses Heuristic Graph Partitioning SWAP* (FM-Graph Partitioning in <a href="#ref-2" class="reference-citation">[2]</a>) is defined similarly to UGPS except for the neighbors of a solution. The neighborhood of a solution may be obtained by a sequence of up to $|V|$ swaps following the Fiduccia-Mattheyses heuristic (see <a href="#ref-2" class="reference-citation">[2]</a>). FM-Graph Partitioning is also $\mathsf{FP}$-complete <a href="#ref-4" class="reference-citation">[4]</a><a href="#ref-2" class="reference-citation">[2]</a>. In fact, a simpler local search problem in which each partition has just one neighbor, the one obtained after the first step of the Fiduccia-Mattheyses heuristic, is also $\mathsf{FP}$-complete <a href="#ref-4" class="reference-citation">[4]</a><a href="#ref-2" class="reference-citation">[2]</a>. This problem is called FM in <a href="#ref-4" class="reference-citation">[4]</a> and FM-SWAP in <a href="#ref-2" class="reference-citation">[2]</a>. In <a href="#ref-4" class="reference-citation">[4]</a> they also show that a simulated annealing based on a SWAP neighborhood and a "cooling schedule" is $\P$-complete. Several additional results are presented proving other graph partitioning strategies are $\P$-hard. See <a href="#ref-5" class="reference-citation">[5]</a><a href="#ref-4" class="reference-citation">[4]</a> for further details.
