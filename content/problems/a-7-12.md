## Given

A sublinear time bounded cellular tree automaton (CTA) $M$ and an input string $x$. See Ibarra, Jiang, and Chang [2] for the definition.

## Problem

Does $M$ accept $x$?

## Status

$\P$-complete (Ibarra, Jiang, and Chang [2]).

## Remarks

A sublinear time one-way bounded CTA can also accept a $\P$-complete language [2]. In a one-way CTA, communication between vertices is bottom-up. Every language accepted by an oneway CTA can be accepted by a deterministic Turing machine in $(\log n)^2 / \log \log n$ space, where $n$ is the size of the input [2].
