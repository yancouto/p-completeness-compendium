## Given

A sublinear time bounded cellular tree automaton (CTA) $M$ and an input string $x$. See Ibarra, Jiang, and Chang [165] for the definition.

## Problem

Does $M$ accept $x$?

## Status

P-complete problem from Greenlaw et al. (1995), Problem A.7.12.

## Remarks

A sublinear time one-way bounded CTA can also accept a P-complete language [165]. In a one-way CTA, communication between vertices is bottom-up. Every language accepted by an oneway CTA can be accepted by a deterministic Turing machine in $(\log n)^2 / \log \log n$ space, where $n$ is the size of the input [165].
