## Given

An $n^\text{th}$ degree polynomial $p \in \Z[x]$, and an integer $\mu$.

## Problem

For each (real or complex) root $z_i$, $1 \leq i \leq n$, of $p$, an approximation $\tilde{z}_i$ such that $|z_i - \tilde{z}_i| < 2^{-\mu}$.

## Status

Open.

## Remarks

If all the roots are real, the problem is in $\NC$ <a href="#ref-1" class="reference-citation">[1]</a>.
