## Given

A finite set $S$, an initial partition $\pi_0$ of $S$, a designated set $S' \subseteq S$, and a set of binary relations $R_1, \ldots, R_k$ on $S$.

## Problem

Is $S'$ a member of the coarsest refinement $\pi$ of $\pi_0$ such that for each pair of blocks $B_1$ and $B_2$ of $\pi$, and for each integer $i = 1, \ldots, k$ either $B_1 \subseteq PI_i(B_2)$ or $B_1 \cap PI_i(B_2) = \emptyset$, where $PI_i(B_2)$ is the pre-image set $PI_i(B_2) = \\{x \mid \text{there is a } y \in B_2 \text{ with } x R_i y\\}$.

## Status

$\P$-complete problem from Greenlaw et al. (1995), Problem A.7.14.

## Remarks

Even if there is only a single relation, $k$ equals one, the problem is $\P$-complete [1]. Sang Cho and Huynh give a $\log n$ time, $O(n^2)$ processor CREW-PRAM algorithm for the single function Coarsest Set Partition Problem as well as a $\log n$ time, $O(n^3)$ processor EREW-PRAM algorithm for the same problem [1]. See also the remarks for Problems A.7.10 and B.4.1.
