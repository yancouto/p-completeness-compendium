## Given

A directed graph $G = (V, E)$ with an ordering on the edges and a designated edge $e$.

## Problem

Is $e$ contained in the edge maximal acyclic subgraph?

## Status

$\P$-complete (Greenlaw <a href="#ref-1" class="reference-citation">[1]</a>).

## Remarks

The reduction to show hardness is from [NORCVP]({{< relref "./a-1-5.md" >}}).

The decision problem "Is the edge maximal subgraph of size $k$?" is also $\P$-complete <a href="#ref-1" class="reference-citation">[1]</a>. The *Feedback Arc Set Problem* (Karp <a href="#ref-2" class="reference-citation">[2]</a>) is an equivalent formulation of the *Maximum Acyclic Subgraph Problem*. Approximation algorithms for this problem are important because there are very few classes of graphs for which the problem is known to be in $\P$ (Berger and Shor <a href="#ref-3" class="reference-citation">[3]</a>). These versions are both $\NP$-complete (Garey and Johnson <a href="#ref-4" class="reference-citation">[4]</a>). Greenlaw proves decision problems based on several other natural approximation algorithms for this problem are $\P$-complete <a href="#ref-1" class="reference-citation">[1]</a><a href="#ref-5" class="reference-citation">[5]</a>. There are other reasonable approximation algorithms that are in $\NC$, see <a href="#ref-1" class="reference-citation">[1]</a><a href="#ref-5" class="reference-citation">[5]</a>. Berger presented an $\NC$ approximation algorithm that, assuming the input graph does not contain two-cycles, generates a subgraph containing more than half the arcs <a href="#ref-6" class="reference-citation">[6]</a>. Ramachandran, Ramachandran proved that finding a minimum weight feedback arc set in a reducible flow graph with arbitrary weights on the arcs is $\P$-complete <a href="#ref-7" class="reference-citation">[7]</a><a href="#ref-8" class="reference-citation">[8]</a>.
