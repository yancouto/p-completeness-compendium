## Given

A compact convex set $M \subseteq \mathbb{R}^n$, a constant $c \in \mathbb{R}^+$, and a norm $\|\cdot\|$ defined by $\|x\| = \max_{1 \leq i \leq n} x_i$ (the maximum norm).

## Problem

Find $x, y \in M$ such that $\|x\| \leq c\|y\|$ for all $y \in M \setminus \\{\vec{0}\\}$. That is, find a vector in $M$ that is lexicographically first among all maximal vectors.

## Status

$\P$-complete (Hoover [1][2]).

## Remarks

This problem can be formulated using a lexicographic ordering function $L(x) = (x, \|x\|)$ to establish a total ordering on vectors in $M$. The problem is to find the lexicographically smallest element according to this ordering among those vectors that maximize the given norm. The result demonstrates that computing lexicographically optimal solutions over continuous domains can be computationally hard even when the underlying optimization problem (finding maximal vectors) might seem tractable.
