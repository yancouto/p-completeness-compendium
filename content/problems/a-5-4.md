## Given

An undirected graph $G = (V, E)$.

## Problem

Find a locally optimal subset of vertices.

A subset is *locally optimal* if it has maximum cost among its neighbors. The *cost* is the sum of the number of edges leaving vertices in the subset; each edge has a weight of one. A *neighbor* of a subset $S$ is a set of size $|S|$ whose symmetric difference with $S$ contains one vertex.

## Status

$\P$-hard or $\FP$-complete (Papadimitriou, Schäffer, and Yannakakis <a href="#ref-3" class="reference-citation">[3]</a>, Schäffer and Yannakakis <a href="#ref-4" class="reference-citation">[4]</a>).

## Remarks

The reduction to show hardness is from [UNAE3SAT]({{< relref "./a-5-3.md" >}}).
The weighted version of the problem in which each edge is given an integer weight encoded in binary is $\mathsf{PLS}$-complete <a href="#ref-3" class="reference-citation">[3]</a><a href="#ref-4" class="reference-citation">[4]</a>. If the edge weights are encoded in unary, the problem is $\mathsf{FP}$-complete. The *Different than Majority Labeling Problem* is equivalent to this problem <a href="#ref-4" class="reference-citation">[4]</a>. A different than majority labeling of an undirected graph $G = (V, E)$ is a labeling of the vertices with either $0$ or $1$ such that each vertex is labeled differently than at least half of its neighbors. Luby describes an $\NC$ algorithm for a related problem <a href="#ref-5" class="reference-citation">[5]</a>. Sarnath and He show that a lexicographic version of the problem is $\P$-complete <a href="#ref-6" class="reference-citation">[6]</a>, see [LFDTML]({{< relref "./a-5-8.md" >}}).
