## Given

A set of pairs of terms $S = \\{(M_1, N_1), \ldots, (M_k, N_k)\\}$ that is *left linear*.

## Problem

Is $S$ *semi-unifiable*?

## Definitions

See [LLSU]({{< relref "./a-6-11.md" >}}).

## Status

Open (Henglein <a href="#ref-1" class="reference-citation">[1]</a>).

## Remarks

2LLSU is $\P$-complete <a href="#ref-1" class="reference-citation">[1]</a>; see [LLSU]({{< relref "./a-6-11.md" >}}). Henglein conjectures that 1LLSU is $\P$-complete <a href="#ref-1" class="reference-citation">[1]</a>.
