## Given

A set of pairs of terms $S = \\{(M_1, N_1), \ldots, (M_k, N_k)\\}$ that is left linear. See Problem A.6.11 for definitions.

## Problem

Is $S$ semi-unifiable?

## Status

Open problem from Greenlaw et al. (1995), Problem B.3.2.

## Remarks

2LLSU is $\P$-complete [1]; see Problem A.6.11. Henglein conjectures that 1LLSU is $\P$-complete [1].
