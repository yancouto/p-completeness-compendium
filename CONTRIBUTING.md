# Contributing to the P-Complete Problems Compendium

Thank you for your interest in contributing!

📖 **Full contributing guidelines are on the website:**  
**[Contributing Guide](https://yancouto.github.io/p-completeness-compendium/about/#contributing)**

## Quick Start

1. **Fork** the repository
2. **Create a problem file** in `content/problems/` using the template on the website
3. **Submit a Pull Request**

## Development Setup

```bash
# Install Hugo (extended version)
# https://gohugo.io/installation/

# Clone and run
git clone https://github.com/yancouto/p-completeness-compendium.git
cd p-completeness-compendium
hugo server -D
# Open http://localhost:1313
```

## Reporting Issues

Found an error? [Open an issue](../../issues) with the problem ID and suggested correction.
